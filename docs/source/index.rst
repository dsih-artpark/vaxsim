VAXSIM
==================

Overview
---------
VAXSIM is a simulation package that implements the SIRSV model with advanced features including Weibull distribution-based immunity waning, targeted vaccination strategies, model calibration tools, and rich visualization options. This documentation provides everything you need to get started, from installation instructions to detailed usage examples.

Core Features

* Random and Targeted Vaccination Strategies
* Model Calibration Tools
* Visualization and Analysis
